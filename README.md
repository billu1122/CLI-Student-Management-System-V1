# CLI-Student-Management-System-V1
My first complete Python application featuring CRUD operations and modular programming.



🎓 Student Management System (Version 1)

📖 About

This project started as a simple Python marks calculator while I was learning the fundamentals of programming.

As I continued learning, I kept improving it instead of creating new small projects. Over time, it evolved into a beginner Student Management System capable of storing student records, calculating marks, editing information, viewing records, and deleting data.

This is my first complete Python application and represents my journey from learning basic Python syntax to understanding modular programming, CRUD operations, and software design.

---

✨ Features

- Add new student records
- Store student information
- Calculate total marks
- Calculate percentage automatically
- View all student records
- View individual student data
- Edit existing student information
- Delete individual student records
- Delete all records
- Input validation for incorrect entries
- Modular project structure using multiple Python files
